Hockey Resource Pack

Experience hockey in Harlon with our hockey resource pack - you will need this to be able to play a game.

Download now

Install on Windows

Right click the Windows icon and press Run.

Write %appdata% and press OK.

Open .minecraft and then resourcepacks, and then drag the folder into the resourcepacks folder. Do NOT unzip the file.

Press F3+T whilst playing Minecraft to reload texture packs, or restart Minecraft.

Install on MacOS

Click on the search button in the MacOS navigation bar.

Write ~/library/Application Support/minecraft, a minecraft folder should then appear below.

Open the minecraft folder and then resourcepacks, and then drag the HHL Texture Pack folder into the kbd>resourcepacks folder. The file should be unzipped.

Press F3+T whilst playing Minecraft to reload texture packs, or restart Minecraft.

On the main menue, click on Options, or whilst in-game click ESC and then Resource Packs. Select the resource pack with the HHL logo and you're good to go!
