Credits

Harlon makes use of many open-source plugins, resources and more to create the best experience for our players. It's only right we credit them on a dedicated page.

Server Plugins
Plugin Name Author Licence Purpose

AdvancedRegionMarket

alex9849

Open-source, but no license specified.

The system powering all shops and properties around the server by allowing WorldGuard regions to be bought and sold.

BKCommonLib

bergerhealer

MIT

A common library for certain plugins, including TrainCarts, SignLink, TCCoasters and LightCleaner.

Brewery

DieReicheErethons

GPL 3.0

Powers the server's brewing system that allows players to brew alcoholic drinks.

BuyCraftX (now known as Tebex)

Tebex Limited (part of Overwolf)

MIT

The plugin which the server store system connects to to execute commands upon purchase (i.e., to allow players who've purchased Donator ranks or particle effects to receive these in-game). Still bears the name of BuyCraftX even though the server store system is now called Tebex.

Christmas Calendar

Phloxz

Closed source/proprietary

Provides the advent calendar system for Christmas. Hooks into the Christmas Presents plugin (see below) to prvide gifts.

Christmas Presents

Phloxz

Closed source/proprietary

Provides Christmas Presents for the above Christmas Calendar plugin.

Citizens (Premium Plugin)

thefullwall

Closed source/proprietary

Provides NPCs for various server purposes, such as Quests and for decoration.

Colored Books

VanillaChai/Chailotl

GPL 3.0

Enables coloured books to be read/written on the server.

Colored Signs

SirBlobman, SwaggerGamer

GPL 3.0

Enables coloured signs to be seen/made on the server.

Command Hook

GPL 3.0

_NewAge/NewAgeCZ

The plugin which the server store system connects to to execute commands upon purchase (i.e., to allow players who've purchased Donator ranks or particle effects to receive these in-game). Still bears the name of BuyCraftX even though the server store system is now called Tebex.

CoreProtect

PlayPro

Artistic 2.0

Provides server protection against griefing, vandalism etc.

DiscordSRV

DiscordSRV Contributors

GPL 3.0

Enables server chat, player join, leave and AFK messages and server messages to be syndicated onto our Discord server.

DiscordSRV-Staff-Chat

DiscordSRV Contributors

MIT

Builds upon DiscordSRV to also provide a staff chat that can be used across both the server and Discord server, with messages being syndicated across the two platforms.

Dynmap®

WebBukkit

Apache 2.0

The plugin that powers the server's dynmap. Harlon does not use the default frontend and instead uses the LiveAtlas as a frontend (see the Miscellaneous resources section) plus icons derived from Google's Material Design Icons (see Miscellaneous section).
